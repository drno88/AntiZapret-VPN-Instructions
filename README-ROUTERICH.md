# Инструкция по настройке AmneziaWG клиента для AntiZapret на Routerich

> Инструкция проверялась на **Routerich** с прошивкой `RouteRich 24.10.4 r28959-29397011cc ASU` и **OpenWRT 24.10.5**. Общий принцип одинаковый для всех версий OpenWRT.

---

## 📋 Требования

- Роутер с прошивкой **OpenWRT 24+** (или совместимой, например Routerich)
- Полученный конфигурационный файл `antizapret.conf`
- Доступ к веб-интерфейсу LuCI и/или SSH

---

## 🔹 Шаг 0: Отключение дополнительных пакетов производителя (Только Routerich)

> ⚠️ **Важно:** Не должно быть установлено **Podkop** / **Zeroblock** — они перестанут работать!

1. Перейдите: **Система** → **Автозапуск**
2. Нажмите **СТОП** и переключите «включено» → «выключено» (интернет может пропасть!)
3. Отключите следующие сервисы:
   - `stubby`
   - `doh-proxy`
   - `dns-failsafe-proxy`
   - `stubby-intercept`
   - `youtubeUnblock`
4. Нажмите **Сохранить**

---

## 🔹 Шаг 1: Отключение DNS Rebinding

1. Откройте веб-интерфейс роутера (LuCI)
2. Перейдите: **Сеть** → **DHCP и DNS** → **Фильтр**
3. Снимите галочку с опции `DNS Rebinding`
4. Поставьте галочку **Фильтровать IPv6-записи типа AAAA**
5. Перейдите во вкладку **«Перенаправление»** и удалите всё в разделе «Перенаправление запросов DNS»
6. Перейдите во вкладку **«Файлы resolv & hosts»**:
   - Поставьте галочку **Использовать /etc/ethers**
   - Снимите галочку **Игнорировать файл resolv**
7. Нажмите **Сохранить и применить**

---

## 🔹 Шаг 2: Установка пакетов AmneziaWG

> Если прошивка от производителя, пакеты обычно уже установлены. Проверьте в любом случае.

1. Откройте **Система** → **Пакеты**
2. Переключитесь на вкладку **«Установлено»** и введите в поиск `Amnezia`
3. Убедитесь, что установлены следующие пакеты:
   - `amneziawg-tools`
   - `kmod-amneziawg`
   - `luci-app-amneziawg`
   - `luci-i18n-amneziawg-ru`

   ![Проверка пакетов AmneziaWG](images/packages_check.png)

4. Если пакеты не установлены — обновите список и установите их:
   ```sh
   opkg update
   opkg install amneziawg-tools kmod-amneziawg luci-app-amneziawg luci-i18n-amneziawg-ru
   ```

5. Перезайдите в веб-интерфейс в режиме **инкогнито** браузера, чтобы появился раздел управления AmneziaWG

---

## 🔹 Шаг 3: Настройка DNS на WAN

1. Перейдите: **Сеть** → **Интерфейсы** → **WAN** → **Изменить**
2. Откройте вкладку **Расширенные настройки**
3. Снимите галочку **Использовать объявляемые узлом DNS-серверы**
4. В поле **Использовать собственные DNS-серверы** пропишите публичные DNS, например:
   - `1.1.1.1`
   - `9.9.9.9`
5. Нажмите **Сохранить и применить**

> ⚠️ Запомните эти DNS! Они понадобятся при редактировании конфигурации AmneziaWG.

---

## 🔹 Шаг 4: Редактирование файла `antizapret.conf`

Откройте файл `antizapret.conf` на ПК и внесите следующие изменения:

1. **Address**: замените маску `/32` на `/24`
2. **DNS**: укажите DNS, который вы назначили на WAN-интерфейс (например, `1.1.1.1`). Либо укажите DNS, которые выдал провайдер автоматически, если вы не меняли DNS на WAN.
   > Смысл: мы будем перехватывать DNS-запросы и направлять их через VPN.
3. **AllowedIPs**: добавьте DNS-серверы вашего провайдера и подсети AntiZapret

### Пример конфигурации:

```ini
[Interface]
PrivateKey = KEY...
Address = 10.29.8.4/24
DNS = 10.29.8.1
Jc = 100
Jmin = 20
Jmax = 100
S1 = 0
S2 = 0
H1 = 1
H2 = 2
H3 = 3
H4 = 4
I1 = ??

[Peer]
PublicKey = KEY...
PresharedKey = KEY...
Endpoint = IP:PORT
AllowedIPs = 1.1.1.1/32, 9.9.9.9/32, 10.29.8.0/24, 10.30.0.0/15, ...
PersistentKeepalive = 15
```

---

## 🔹 Шаг 5: Импорт конфигурации через Web-UI

1. Перейдите: **Сеть** → **Интерфейсы** → **Создать новый интерфейс** → тип `AmneziaWG`
2. Вставьте содержимое отредактированного файла `antizapret.conf`
3. Перейдите во вкладку **«Равноправные узлы»**, нажмите **Изменить** на добавленной конфигурации
4. Прокрутите вниз и поставьте галочку **Маршрутизировать разрешённые IP-адреса**
5. Нажмите **Сохранить**

---

## 🔹 Шаг 6: Настройка фаервола и маршрутизации

### Зоны фаервола

1. Перейдите: **Сеть** → **Межсетевой экран**
2. Создайте новую зону, например **Antizapret** (название может быть любым)
3. Настройте следующие параметры:
   - ✅ **Маскарадинг (Masquerading)**
   - ✅ **Ограничение MSS**
   - **Охватываемые сети**: выберите созданный интерфейс AmneziaWG
   - **Разрешить перенаправление в «зоны назначения»**: `WAN`
   - **Разрешить перенаправление из «зон источников»**: `WAN` + `LAN`
4. Нажмите **Сохранить**

### Перезагрузка

Нажмите **Сохранить и применить** в LuCI, затем **перезагрузите роутер**.

---

## 🔹 Шаг 7: Проверка работы

1. Откройте LuCI → **Сеть** → **Интерфейсы** и убедитесь, что на интерфейсе AmneziaWG идут пакеты (как принятые, так и отправленные)
2. На ПК откройте командную строку (`cmd`) и проверьте пинг до заблокированных ресурсов, например `youtube.com`
   - Вы должны получать ответ из подсети `10.29.8.0/24` или `10.30.0.0/15`, а не реальный публичный IP
3. То же самое можно проверить с самого роутера через SSH:
   ```bash
   ping youtube.com
   ```
