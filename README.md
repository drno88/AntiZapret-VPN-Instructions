# 🌐 AntiZapret VPN — Инструкции по настройке OpenWRT

> Сборник инструк по настройке клиента **AntiZapret VPN** на роутерах с прошивкой **OpenWRT**.

Проект [AntiZapret VPN](https://github.com/GubernievS/AntiZapret-VPN) — это бесплатная система обхода блокировок для российских пользователей.

---

## 📖 Доступные инструкции

| Инструкция | Описание |
|---|---|
| [Routerich / OpenWRT — AmneziaWG](README-ROUTERICH.md) | Настройка AmneziaWG на Routerich и OpenWRT 24+ |
| [WireGuard — OpenWRT](README-WIREGUARD.md) | Настройка WireGuard на OpenWRT 23.05.5+ |

---

## 🚀 Быстрый старт

1. Получите конфигурационный файл на сайте [AntiZapret](https://antizapret.prostovpn.org/) или в [Telegram-боте](https://t.me/antizapret_vpn_bot)
2. Выберите инструкцию для вашего типа подключения и модели роутера
3. Следуйте шагам из инструкции

---

## 📋 Требования

- Роутер с прошивкой **OpenWRT 23+** (или совместимой, например Routerich)
- Доступ к веб-интерфейсу LuCI и/или SSH
- Конфигурационный файл от AntiZapret
